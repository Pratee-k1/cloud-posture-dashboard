# Cloud Posture Dashboard

## Overview
This project demonstrates a simple Cloud Security Posture Management (CSPM) dashboard
that allows users to view and filter cloud misconfigurations across multiple cloud providers.

## Features
- Multi-cloud visibility (AWS, Azure, GCP)
- Severity-based risk prioritization
- Compliance mapping (PCI-DSS, SOC2, CIS, ISO27001)
- Interactive filtering
- Clean and intuitive UI

## How to Run
Open `src/frontend/dashboard.html` in a browser
OR use VS Code Live Server.

## Design Approach
- Normalized misconfiguration data model
- Client-side filtering for simplicity
- Risk-first dashboard design
